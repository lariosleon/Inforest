-----------------------
System Requirement
-----------------------
SecuGen USB port devices based on FDU02, FDU03/SDU03, FDU04/SDU04, or U20:
- IBM-compatible PC 486 or later
- CD ROM Drive
- 1 USB 2.0 port 
- 64MB RAM
- 80MB available hard disk space
- Microsoft Windows 98 SE, Windows ME, Windows 2000/XP/2003, Windows Vista,
  Windows 7,8,8.1
